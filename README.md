# drakk3ai

Personal platform.

## App stack

- **Expo React Native** — cross-platform mobile framework
- **Expo Router** — file-based navigation
- **TypeScript** — strict typing throughout
- **Zustand** — global state management (slices per feature)
- **React Query** — server state and cache layer
- **Supabase** — auth, PostgreSQL, Edge Functions, Realtime
- **react-native-maps** — Google Maps / Apple Maps rendering
- **expo-location** — GPS tracking
- **expo-task-manager** — background location task
- **expo-notifications** — push notifications via FCM and APNs
- **expo-haptics** — haptic feedback on zone events

## AI stack

- **Claude Code** — AI coding assistant (CLI)
- **Engram** — persistent memory across sessions
- **GGA** — pre-commit AI code review
- **SDD** — spec-driven development workflow
